# Manahil Iqbal — Notebook Portfolio (React)

A notebook-style portfolio built with React and Vite, converted from the original HTML mockup.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

The static site output is in `dist/`.

## Project structure

- `src/components/NotebookPortfolio.jsx` — main layout and sections
- `src/data/portfolioData.js` — content (skills, experience, projects, etc.)
- `src/styles/NotebookPortfolio.css` — notebook visual styles
- `manahil_notebook_portfolio.html` — original HTML reference
